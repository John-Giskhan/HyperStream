# HyperStream

Turn your personal media library into a fast, private, cross-device streaming service.

HyperStream is a self-hosted media platform built for one goal: start playback fast in the highest possible quality currently available to your device.

## Features

- Direct stream first for maximum quality and near-zero server overhead.
- Adaptive fallback when a client cannot play the source codec/container.
- High-performance C++ server core designed for low latency and high throughput.
- Multi-client experience across Chromecast, Fire TV, Apple TV, Android, iOS, and Web.
- Self-hosted and private: your media, your hardware, your rules.
- Support for AV1/H265(HEVC)/VP9/H264 video codecs with Dolby Vision/HDR10.
- Support for TrueHD (with Atmos)/FLAC/EAC3/AC3/AAC.
